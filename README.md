# weather-app
# Instructions 🌨

1. Type the city in the search box, Click 'Search' 🔍
2. The following will be displayed:
    - Current temperature
    - Highest temperature
    - Lowest temperature
    - Humidity
    - Weather conditions
## Project setup
```
npm install
```
### Compiles and hot-reloads for development
```
npm run serve
```
##### <u>API Reference</u>
source: <a href="https://openweathermap.org/">Open Weather</a>
